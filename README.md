# Banana_Bot

基于 Arduino 的四轮麦克纳姆轮全向移动底盘 + 6自由度机械臂遥控机器人。

## 项目简介

Banana_Bot 是一台面向机械赛的遥控机器人，具备全向移动和物体抓取能力。整车由两块 Arduino 分别控制底盘和机械臂，通过串口通信协调工作，操作员使用 SBUS 遥控器进行控制。

### 主要特性

- 麦克纳姆轮全向移动（前后、横移、自转）
- 6自由度机械臂，支持多种预设抓取/放置动作
- SBUS 遥控协议，低延迟高可靠
- 遥控器失联自动停车保护（Failsafe）

## 软件架构

项目包含两个独立的 Arduino 程序，分别运行在底盘主控和机械臂主控上。

### 底盘控制系统 (`main/`)

```
main/
├── main.ino      # 入口：初始化 + 主循环
├── include.h     # 统一头文件引用
├── App.cpp/.h    # 任务调度与遥控信号解算
├── motor.cpp/.h  # 电机驱动、编码器反馈、PID 控制
└── signal.cpp/.h # SBUS 协议接收与机械臂指令转发
```

**控制流程：**

```
loop()
 └─> TaskRun()                    [30ms 周期]
      ├─> Write()                 向机械臂发送控制指令
      ├─> GetSignal()             读取 SBUS 遥控器数据
      ├─> graph()                 串口输出调试曲线
      └─> 信号解算
           ├─> failsafe?  ──yes──> stop_all()    紧急停车
           ├─> 油门低/摇杆居中? ──> stop_all()    待机
           └─> 正常控制:
                ├─> ch[1] → 前后速度 (mafor~mdfor)
                ├─> ch[0] → 横移速度 (mago~mdgo)，对角轮反向
                ├─> ch[3] → 自转速度 (maturn~mdturn)，左右侧反向
                └─> 三轴叠加 → ref1~ref4 → setspeed() → moving()
```

**麦克纳姆轮运动学：**

| 运动方向 | M1(左前) | M2(右前) | M3(左后) | M4(右后) |
|---------|---------|---------|---------|---------|
| 前进     | +       | +       | +       | +       |
| 横移(右) | +       | -       | +       | -       |
| 自转(顺) | +       | +       | -       | -       |

最终每个电机的输出 = 前后分量 + 横移分量 + 自转分量，限幅 ±255。

**PID 控制器参数：**
- Kp = 0.6, Ki = 0, Kd = 0（当前仅使用比例控制）
- 编码器采样周期：30ms

### 机械臂控制系统 (`arm code/main/`)

```
arm code/main/
└── main.ino      # 6自由度舵机控制，含串口指令解析与速度插值
```

**控制流程：**

```
loop()
 ├─> GetData()                    解析串口指令 [角度, 动作编号, 夹爪, 辅助]
 ├─> update_state()               根据动作编号查表 → 更新目标角度
 ├─> ServoSetPluseAndTime()       设置目标脉宽与运动时间
 └─> ServoPwmDutyCompare()        每 20ms 步进插值，平滑到达目标位置
```

**预设动作表：**

| 编号 | 动作   | 舵机1 (底座旋转) | 舵机2 (大臂) | 舵机3 (小臂) |
|-----|--------|-----------------|-------------|-------------|
| 0   | 默认    | 500             | 500         | 500         |
| 1   | 抓球    | 2500            | 1000        | 2000        |
| 2   | 放木块  | 1900            | 600         | 2000        |
| 3   | 投球    | 1200            | 1000        | 1000        |
| 4   | 抓木块  | 2500            | 1200        | 1600        |

**舵机速度控制：**
- 运动时间可调（20ms ~ 30000ms），默认 5000ms
- 采用线性插值，每 20ms 步进一次，确保平滑运动
- PWM 脉宽范围：500μs ~ 2500μs

### 通信协议

底盘主控通过 Serial3 (9600bps) 向机械臂发送 JSON 风格数组指令：

```
[旋转角度, 预设动作编号, 夹爪角度, 辅助角度]
```

示例：
```
[1500,1,1500,1200]    # 底座居中，执行"抓球"动作，夹爪1500μs，辅助1200μs
[1500,0,500,1200]     # 底座居中，默认姿态，夹爪全闭，辅助1200μs
```

遥控器通道映射：
| 遥控器通道 | 功能 | 映射方式 |
|-----------|------|---------|
| ch[0]     | 底盘横移 | 340-1704 → PWM ±255 |
| ch[1]     | 底盘前后 | 340-1704 → PWM ±255 |
| ch[2]     | 油门/使能 | <800 时停车 |
| ch[3]     | 底盘自转 | 340-1704 → PWM ±255 |
| ch[4]     | 机械臂辅助轴 | 340-1704 → 500-2500μs |
| ch[5]     | 机械臂旋转 | 340-1704 → 500-2500μs |
| ch[6]     | 夹爪开合 | <1000: 1500μs / 否则: 500μs |
| ch[7]     | 动作模式切换 | <1000: 投球模式(3) |
| ch[8]     | 动作细选 | <1000: 抓球(1) / <1300: 默认(0) / 否则: 放木块(2) |

## 编译与烧录

1. 安装 [Arduino IDE](https://www.arduino.cc/en/software) (≥1.8)
2. 通过 Library Manager 安装上述依赖库
3. **底盘程序**：打开 `main/main.ino` → 板型选择 `Arduino Mega 2560` → 编译烧录
4. **机械臂程序**：打开 `arm code/main/main.ino` → 板型选择对应 Arduino 板 → 编译烧录
5. 接线完成后上电，遥控器对频即可操控
