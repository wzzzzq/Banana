# Banana - 麦克纳姆轮机械臂小车

基于 Arduino 的四轮麦克纳姆轮底盘 + 6自由度机械臂遥控机器人项目。

## 项目简介

本项目为机械赛参赛作品，包含两个独立的 Arduino 控制系统：

- **底盘控制系统** (`main/`)：通过 SBUS 遥控器控制四轮麦克纳姆轮底盘，支持全向移动
- **机械臂控制系统** (`arm code/main/`)：6自由度舵机机械臂，支持预设位置切换和视觉追踪

## 系统架构

```
遥控器 (SBUS) ──> 底盘主控 (Arduino Mega) ──Serial3──> 机械臂主控 (Arduino)
                      │                                      │
                 4x 直流电机                            6x 舵机
                 (麦克纳姆轮)                         (Learm 机械臂)
```

## 依赖库

- [Servo](https://www.arduino.cc/reference/en/libraries/servo/) - Arduino 自带
- [SBUS](https://github.com/bolderflight/sbus) - Bolder Flight Systems SBUS 协议库
- [PID_v1](https://github.com/br3ttb/Arduino-PID-Library) - PID 控制库

## 编译与烧录

1. 安装 [Arduino IDE](https://www.arduino.cc/en/software)
2. 通过 Library Manager 安装上述依赖库
3. 底盘程序：打开 `main/main.ino`，选择 **Arduino Mega 2560** 板型，编译烧录
4. 机械臂程序：打开 `arm code/main/main.ino`，选择对应板型，编译烧录

## 通信协议

底盘主控通过 Serial3 向机械臂发送控制指令，格式为：

```
[旋转角度, 预设动作编号, 夹爪角度, 辅助角度]
```

- 预设动作编号：0=默认, 1=抓球, 2=放木块, 3=投球, 4=抓木块
- 角度值范围：500-2500（对应舵机 PWM 脉宽，单位 μs）
